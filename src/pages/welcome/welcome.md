  11ty uses the command line to get up and running, which is a bit intimidating at fist, but once it it up and running it is easy to add on to.

If you're looking for something more advanced, they also have a collection of [starter projects](https://www.11ty.dev/docs/starter/) that have a larger codebase with some more advanced layout and styling.
